# `prereg/notes/`

Working notes feeding the pre-registration. Files here are scratch — drafts, cross-bot reviews, synthesis worksheets — and may not be self-consistent until promoted into `prereg/prereg.md`.

## Files

- **`p3-cross-bot-prompt.md`** — canonical standardised prompt for parallel independent review of the B1/B2 operationalisation across multiple bot families. Dispatch instructions at the top of the file.
- **`p3-claude-draft.md`**, **`p3-gemini-draft.md`**, **`p3-gpt5-draft.md`** — drop-in stubs for each bot's response. Fill from fresh chat sessions, no prior context.
- **`p3-synthesis.md`** *(not yet written)* — Jon's synthesis of the three drafts; the bridge between this directory and the prereg's "Recursive OOD probe" subsection.

## Convention

Bot drafts are quoted verbatim, not paraphrased. The synthesis file extracts load-bearing sentences with attribution. Per `../../CLAUDE.md` "How to challenge well": this directory is where disagreement between bots gets surfaced and worked, before any conclusion is moved into the prereg.
